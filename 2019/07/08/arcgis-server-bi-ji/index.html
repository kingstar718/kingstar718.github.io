ArcGIS Server 笔记

Posted on 2019-07-08 | In 学习笔记

一丶创建服务

  1. ->打开ArcMap
  2. ->目录->GIS服务器
  3. ->添加ArcGIS Server
  4. ->管理GIS服务器
  5. ->输入服务器URL丶用户名密码(即ArcGIS Server登录名密码)
  6. ->在目录中找到创建好的服务器,新建文件夹
  7. ->点击ArcMap的’文件’->共享为->服务
  8. ->选择发布服务->选择刚才建好的连接
  9. ->将服务发布至文件夹选择刚才新建的文件夹
  10. ->进入服务编辑器,进行相应选择->发布服务

二、问题解决

问题1: 如果显示发布失败,检查服务器工具之类的,打开ArcMap的服务页面(../manager)，点击System,选择PublishingTools，并点击保存并重新发布,如果发布失败，显示端口4000被占用，打开http://localhost:6080/arcgis/admin/machines/这里是您的机器名/edit，选择把4000端口改为4006或4005,重启PublishingTools，再回到ArcMap发布服务

问题2: 解决ArcGIS Service中的服务在内网环境下无法进行javascript预览问题。
在使用ArcGIS for Service时经常会遇到一个问题，那就是我们需要对已经发布的服务进行预览，预览时点击对应服务，选择View in中的ArcGIS JavaScript就可在浏览器中实现对地图的预览工作。
原因：Service中配置的JavaScript API地址为在线服务地址，内网会导致网络不通。
解决:登陆后台管理： http://localhost:6080/arcgis/admin/login
后台管理中找到以下路径：http://localhost:6080/arcgis/admin/system/handlers/rest
点击servicesdirectory->点击edit,修改API URL和CSS URL以及CSS2 URL
